## Background

This project explored content based image retrieval using convolutional neural networks, with a focus on Siamese architectures and one-shot learning.

Content based image retrieval extracts visually similar images from a database based on the features of a query image. The project used a Siamese neural network trained on pairs of dissimilar images to produce feature vectors. Similarity was then estimated using distances between learned embeddings.

## Datasets

The work used the Omniglot dataset and a modified version of the Stanford Dogs dataset.
